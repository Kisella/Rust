# 0-1背包问题
## 1 题目
给定一个商品列表，包括商品名、价格和体积信息;再给定一个体积容量为`capacity`的背包，允许从商品列表中选择一件或多件不同的商品带走。问，如何才能带走总价最多的商品？
```
    ------------------------
      商品     价格     体积
    ------------------------
      啤酒     24       10
      汽水     2        3
      饼干     9        4
      面包     10       5
      牛奶     9        4
      葡萄     4        1
    ------------------------
```
&nbsp;
## 2 思路
### 2.1 拆解子问题，声明动态规划数组`dp[]`
声明动态规划数组`dp[]`, 长度为背包容量`capacity` + 1, 初始化为零
&nbsp;
### 2.2 动态规划
- 第一层迭代：遍历商品列表
- 第二层迭代：从背包容量反向遍历到商品的体积
- 递推关系式`dp[c] = max(dp[c], dp[c - item.volume) + item.price`
```rust
    for item in commodity_list.iter() {
        for c in (item.volume..=capacity).rev() {
            dp[c] = max(dp[c], dp[c - item.volume] + item.price)    //  递推公式
        }
    }
```
&nbsp;
#### 2.3 回溯法得到选择结果
```rust
    let mut remaining = capacity;
    let mut selected = vec![];
    for (i, item) in commodity_list.iter().enumerate() {
        if remaining >= item.volume && dp[remaining] == dp[remaining - item.volume] + item.price {
            selected.push(i);
            remaining -= item.volume;
        }
    }
```
&nbsp;
### 2.4 递推结束，输出结果
`dp[capacity]`即为背包容量为`capacity`时能够带走商品的最大价值。
